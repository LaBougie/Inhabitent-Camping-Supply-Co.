# Inhabitent

Inhabitent Site is a multi-page website for a Inhabitent Camping Supply Co. Its objective is using WordPress as a content management system.


### Installation

* node v6.4.0
* npm v-3.10.3
* gulp v-4.0.0 


### Technologies

* HTML
* SASS/SCSS
* CSS
* Javascript
* jQuery
* Gulp
* PHP


### Wordpress Plugins

| Askimet |
| Contact Form 7 |
| Custom Field Suite |
| Debug Bar |
| Business Hour Widget |
| Inhabitent Functionality |
| Theme Check |
| WP Migrate DB |
